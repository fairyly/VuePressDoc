# 1.3.3 react 受控组件和非受控组件

>>每当表单的状态发生变化时，都会被写入到组件的 state 中，这种组件在React 中被称为受控组件（controlled component）。    
    在受控组件中，组件渲染出的状态与它的 value 或 checked prop 相对应;   
    如果一个表单组件没有 value props（单选按钮和复选框对应的是 checked prop）时，就可以称为非受控组件;    
    不提倡在 React 中使用非受控组件
